Auction App

A live, browser-based player auction system — auctioneer on one screen, the entire room watching in real time.

What is it?

One app, two views

🖥️

Console Host only

The auctioneer's control panel. Record sales, undo mistakes, change themes, export results — all from one screen.

📺

Viewer Board Everyone

A live, read-only scoreboard that updates instantly on every phone, laptop, or TV in the room. No app needed — just a browser.

⚡

Real-time sync

Updates reach all viewers within milliseconds using Server-Sent Events. No refreshing required.

Auctioneer Console

Everything the auctioneer needs

Header

  • Now Bidding — current player on the table
  • LAN IP — address viewers open on their devices
  • Viewer count — live number of connected screens
  • Show Rules — toggles overlay on all screens
  • Theme — Court / Bosch / Stage

Sale Panel

  • Choose Team and Player from dropdowns
  • Enter winning Tokens amount
  • Hit Confirm Sale — validated instantly
  • Undo reverts the last sale
  • Export downloads a CSV of all results
  • Reset clears all data (auto-backed up)

The console is only accessible from the auctioneer's machine (localhost). All other devices are automatically redirected to the read-only viewer.

Viewer Board

What the audience sees

Main Board

  • Now Bidding — updates the moment a player is selected
  • Sale Ticker — scrolling log of recent sales
  • Team Cards — roster, captain, budget bar per team

Sidebar

  • All Players — full pool; sold players crossed out
  • Rules Overlay — full-screen pop-up triggered by auctioneer
  • Theme — follows the auctioneer's theme choice live

Open http://<LAN-IP>:8080 on any browser on the same Wi-Fi — no install, no login.

Auction Rules

How the bidding works

Team Size

1 captain (pre-assigned) + 3 players via auction = 4 members total. All must play.

Token Purse

Each team starts with 20,000 tokens. Unspent tokens have no value after the auction.

Base Prices

Beginner players start at 2,000 tokens. Intermediate players start at 5,000 tokens. No bids below base price.

Gender Quota

Every team must have min 1, max 2 female members. The captain counts toward this quota.

Last-Slot Rule

If a team (with a male captain) reaches their last auction slot with no female player yet, that slot must go to a female.

Budget Discipline

Teams must keep enough tokens to fill all remaining slots. The app enforces this automatically.

Getting started

Up and running in 2 steps

# 1. Install the only dependency
pip install flask

# 2. Start the auction server
python server.py

# Browser opens automatically at localhost:8080
# Share the LAN IP shown on-screen with viewers
  • Edit players.json and teams.json before the auction to set your roster
  • Hit Reset on the console to reload data files (current state is backed up)
  • Export results as CSV anytime from the console
